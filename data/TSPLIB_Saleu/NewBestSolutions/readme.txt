The new best solutions found in 

M. Dell'Amico, R. Montemanni, S. Novellani, "Matheuristic algorithms for the parallel drone scheduling traveling salesman problem"

are contained in this folder.

The file "pds-40-3-4-2-1-gr229.out" refers to the instance gr229 with 40% of the nodes being drone-eligible, three drones, truck speed 3, drone speed 2, depot location 1.

Each file contains the cost of the relative solution and the list of active variables according to the definition of model PDS. Quantities in brackets are not significant and have to be ignored.
